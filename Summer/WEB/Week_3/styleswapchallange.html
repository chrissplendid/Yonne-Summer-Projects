Style Swap Challenge
Inline vs <style> vs <link>
External CSS

Global theme from <link rel="stylesheet">

This card inherits its look from the external stylesheet included via a data URL. It sets the base :root variables (colors, radius, shadows) and the layout styles.

Base variables Grid layout Card hover
Internal <style>

Scoped overrides in the page

Inside .internal-theme, the page’s internal stylesheet tweaks variables and borders to create a lavender vibe. This demonstrates how internal CSS can override the external file for selected sections.

Scoped theme Custom shadow Border accent
Inline Style

Inline styles win

This card uses the element’s style="..." attribute to override both external and internal CSS. Inline styles carry the highest specificity in the cascade.

Immediate override No selector needed Highest priority

Tip: Order of power → External (link) < Internal (<style>) < Inline (style attribute).
